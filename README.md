## ⚙️ 개발 환경 정보 (Environment)

| 항목              | 상세 내용                 |
| :---------------- | :------------------------ |
| **프레임워크**    | **React 19 + Vite 7**     |
| **언어**          | **TypeScript 5.8**        |
| **스타일링**      | **Styled-Components 6.1** |
| **상태 관리**     | **Jotai 2.15**            |
| **API 통신**      | **Axios 1.12**            |
| **패키지 매니저** | **npm**                   |

---

## 📝 커밋 메시지 작성 규칙

- 작성 방법 : ```[카테고리]``` 작업 내용 요약

|카테고리	|설명|
|--|--|
|추가|	새로운 기능 추가
|수정|	기존 기능 수정, 버그 수정
|삭제|	불필요한 내용 삭제, 기능 삭제
|문서|	코드 편집X, 관련 문서 업로드 할 때
|테스트|	테스트 코드 작업 시
|환경|	빌드, 설정 파일 등 수정, DB 연결 작업

---

## 💻 실행 방법 (Getting Started)

```bash
1. 레포지토리 복사
git clone https://github.com/YU-AIProject-26/Frontend.git

2. 프로젝트 폴더로 이동
cd Frontend

3. 의존성 패키지 설치
npm install

4. 로컬 개발 서버 실행
npm run dev
```
